# bsp-dd

This script is designed to a common use of dynamic desktops, on a single monitor (a laptop for example). It keeps it simple and reliable, but also easy to customize to your habits. It reorders the desktops and remove the empty ones automatically.

## You can

- **Add desktops**

- **Remove desktops**

- **Cycle throught desktops**

- **Quickly move desktops (reorder them)**
